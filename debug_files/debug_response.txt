---ЗАДАЧА---

Прямая, проведённая параллельно боковой стороне трапеции через конец меньшего основания, равного 6, отсекает треугольник, периметр которого равен 18. Найдите периметр трапеции.

---РЕШЕНИЕ---

1. Обозначим трапецию как $ABCD$, где $AB$ — меньшее основание, $CD$ — большее основание, $AD$ и $BC$ — боковые стороны.
2. Пусть прямая, проведённая через точку $B$ параллельно $AD$, пересекает $CD$ в точке $E$. Тогда треугольник $BCE$ имеет периметр 18.
3. Так как $BE$ параллельна $AD$, то $BE = AD$.
4. Периметр треугольника $BCE$ равен $BC + CE + BE$.
5. Периметр трапеции $ABCD$ равен $AB + BC + CD + AD$.
6. Заметим, что $CD = CE + ED$, а $ED = AB$, так как $AB$ и $ED$ — параллельные отрезки между параллельными прямыми $AD$ и $BE$.
7. Подставляя $BE = AD$ и $ED = AB$ в выражение для периметра трапеции, получаем:
$AB + BC + CD + AD = AB + BC + (CE + ED) + AD = AB + BC + CE + AB + AD = 2AB + BC + CE + AD$.
8. Учитывая, что $AB = 6$, а периметр треугольника $BCE$ равен 18, получаем:
$2AB + BC + CE + AD = 2 \cdot 6 + 18 = 30$.

Ответ: $30$

---ПОДСКАЗКИ---

1. В трапеции $ABCD$ с меньшим основанием $AB$ проведена прямая $BE$ параллельно боковой стороне $AD$. Эта прямая отсекает треугольник $BCE$ с периметром 18. Важно помнить, что: 1) $BE$ параллельна $AD$, поэтому $BE = AD$; 2) $AB$ и $ED$ — параллельные отрезки между параллельными прямыми $AD$ и $BE$, поэтому $ED = AB$.
2. Для решения задачи можно использовать свойства параллельных прямых и отрезков. Так как $BE$ параллельна $AD$, то $BE = AD$. Также $AB$ и $ED$ — параллельные отрезки между параллельными прямыми, поэтому $ED = AB$. Периметр треугольника $BCE$ равен $BC + CE + BE$, а периметр трапеции $ABCD$ равен $AB + BC + CD + AD$.
3. План решения задачи:
- Шаг 1: Обозначить трапецию $ABCD$ и прямую $BE$.
- Шаг 2: Использовать свойства параллельных прямых и отрезков для установления равенств между сторонами.
- Шаг 3: Выразить периметр трапеции через известные величины.
- Шаг 4: Подставить известные значения и вычислить периметр трапеции.
