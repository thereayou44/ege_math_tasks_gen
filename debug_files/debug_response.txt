---ЗАДАЧА---

Установите соответствие между графиками линейных функций и угловыми коэффициентами прямых.

ГРАФИКИ:
А) График функции $y = 2x + 1$.
Б) График функции $y = -3x + 4$.
В) График функции $y = \frac{1}{2}x - 2$.
Г) График функции $y = -x + 3$.

УГЛОВЫЕ КОЭФФИЦИЕНТЫ:
1) $2$
2) $-3$
3) $\frac{1}{2}$
4) $-1$

Запишите в ответ цифры, расположив их в порядке, соответствующем буквам: AБВГ.

---РЕШЕНИЕ---

1. Угловой коэффициент прямой определяется коэффициентом при $x$ в уравнении функции вида $y = mx + b$, где $m$ — угловой коэффициент.

2. Для графика А: $y = 2x + 1$, угловой коэффициент $m = 2$.

3. Для графика Б: $y = -3x + 4$, угловой коэффициент $m = -3$.

4. Для графика В: $y = \frac{1}{2}x - 2$, угловой коэффициент $m = \frac{1}{2}$.

5. Для графика Г: $y = -x + 3$, угловой коэффициент $m = -1$.

Ответ: 1234

---ПОДСКАЗКИ---

1. Угловой коэффициент прямой — это число, которое стоит перед $x$ в уравнении прямой вида $y = mx + b$. Он показывает, насколько быстро функция возрастает или убывает. В данной задаче нужно сопоставить каждый график с соответствующим угловым коэффициентом.

2. Посмотрите на уравнения функций, представленные для каждого графика. Угловой коэффициент — это коэффициент при $x$. Например, для функции $y = 2x + 1$ угловой коэффициент равен $2$.

3. Пройдите по каждому графику и определите его угловой коэффициент:
- Для графика А: $y = 2x + 1$, угловой коэффициент $m = 2$.
- Для графика Б: $y = -3x + 4$, угловой коэффициент $m = -3$.
- Для графика В: $y = \frac{1}{2}x - 2$, угловой коэффициент $m = \frac{1}{2}$.
- Для графика Г: $y = -x + 3$, угловой коэффициент $m = -1$.
Теперь сопоставьте эти коэффициенты с предложенными вариантами.
