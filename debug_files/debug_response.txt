---ЗАДАЧА---

Решите уравнение $\sin(2x) = \frac{1}{2}$ на интервале $[-\pi; \pi]$. В ответе укажите все корни уравнения, разделив их запятой.

---РЕШЕНИЕ---

1. Запишем уравнение: $\sin(2x) = \frac{1}{2}$.
2. Определим значения $2x$, при которых $\sin(2x) = \frac{1}{2}$. Это углы $\frac{\pi}{6}$ и $\frac{5\pi}{6}$, а также их периодические повторения.
3. Решим уравнение для $2x$:
   $2x = \frac{\pi}{6} + 2k\pi$ или $2x = \frac{5\pi}{6} + 2k\pi$, где $k \in \mathbb{Z}$.
4. Разделим обе части каждого уравнения на 2:
   $x = \frac{\pi}{12} + k\pi$ или $x = \frac{5\pi}{12} + k\pi$.
5. Найдём корни, принадлежащие интервалу $[-\pi; \pi]$:
   - Для $x = \frac{\pi}{12} + k\pi$:
     $k = -1$: $x = \frac{\pi}{12} - \pi = -\frac{11\pi}{12}$.
     $k = 0$: $x = \frac{\pi}{12}$.
     $k = 1$: $x = \frac{\pi}{12} + \pi = \frac{13\pi}{12}$ (не принадлежит интервалу).
   - Для $x = \frac{5\pi}{12} + k\pi$:
     $k = -1$: $x = \frac{5\pi}{12} - \pi = -\frac{7\pi}{12}$.
     $k = 0$: $x = \frac{5\pi}{12}$.
     $k = 1$: $x = \frac{5\pi}{12} + \pi = \frac{17\pi}{12}$ (не принадлежит интервалу).
6. Корни уравнения на интервале $[-\pi; \pi]$: $-\frac{11\pi}{12}, -\frac{7\pi}{12}, \frac{\pi}{12}, \frac{5\pi}{12}$.

Ответ: $-\frac{11\pi}{12}, -\frac{7\pi}{12}, \frac{\pi}{12}, \frac{5\pi}{12}$

---ПОДСКАЗКИ---

1. В уравнении $\sin(2x) = \frac{1}{2}$ требуется найти все корни на интервале $[-\pi; \pi]$. Обратите внимание, что аргумент синуса — $2x$, а не $x$, поэтому необходимо учесть периодичность и диапазон значений $2x$.

2. Для решения уравнения $\sin(2x) = \frac{1}{2}$ используйте следующие свойства и формулы:
   - Значения синуса $\frac{1}{2}$ соответствуют углам $\frac{\pi}{6}$ и $\frac{5\pi}{6}$.
   - Уравнение $\sin(2x) = \frac{1}{2}$ имеет решения вида $2x = \frac{\pi}{6} + 2k\pi$ или $2x = \frac{5\pi}{6} + 2k\pi$, где $k \in \mathbb{Z}$.
   - Чтобы найти $x$, разделите обе части уравнения на 2.
   - Учтите интервал $[-\pi; \pi]$ при выборе значений $k$.

3. Для решения уравнения $\sin(2x) = \frac{1}{2}$ на интервале $[-\pi; \pi]$ выполните следующие действия:
   - Запишите уравнения для $2x$: $2x = \frac{\pi}{6} + 2k\pi$ или $2x = \frac{5\pi}{6} + 2k\pi$.
   - Разделите обе части каждого уравнения на 2: $x = \frac{\pi}{12} + k\pi$ или $x = \frac{5\pi}{12} + k\pi$.
   - Найдите корни, принадлежащие интервалу $[-\pi; \pi]$, подставляя различные целые значения $k$.
   - Запишите все найденные корни в ответе.
