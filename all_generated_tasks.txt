---ЗАДАЧА---
На графике тригонометрической функции $y = a \cdot \cos(b \pi x + c) + d$ видны следующие особенности: максимальное значение функции равно 2, минимальное — равно -2, и при $x = 0$ значение функции равно 2. Также известно, что период функции равен 4. Найдите значение функции при $x = \frac{100}{5}$.

---РЕШЕНИЕ---
Из графика видно, что $f_{max} = 2$, $f_{min} = -2$. Тогда:
\[
d = \frac{f_{max} + f_{min}}{2} = \frac{2 + (-2)}{2} = 0,
\]
\[
|a| = \frac{f_{max} - f_{min}}{2} = \frac{2 - (-2)}{2} = 2.
\]
Далее, из графика получаем, что $f(0) = 2$. Если $a = 2$, то:
\[
2 \cos(c) = 2 \implies \cos(c) = 1 \implies c = 2k\pi, k \in \mathbb{Z}.
\]
Так как $c$ должно быть целым числом, выбираем $c = 0$.

Наименьший положительный период этой функции равен $\frac{2}{|b|}$, а по графику наименьший положительный период равен 4, следовательно:
\[
\frac{2}{|b|} = 4 \implies |b| = \frac{1}{2}.
\]
Таким образом, $b = \pm \frac{1}{2}$. Окончательно, функция имеет вид:
\[
f(x) = 2 \cos\left(\frac{\pi x}{2}\right).
\]
Найдём $f\left(\frac{100}{5}\right)$:
\[
f\left(\frac{100}{5}\right) = 2 \cos\left(10 \pi\right) = 2 \cdot 1 = 2.
\]

Ответ: $2$

---ПОДСКАЗКИ---
1. В задаче требуется найти значение тригонометрической функции при заданном $x$. Функция имеет вид $y = a \cdot \cos(b \pi x + c) + d$, где $a$, $b$, $c$, $d$ — параметры, которые нужно определить из условия задачи.
2. Для решения задачи используйте следующие свойства тригонометрических функций:
   - максимальное и минимальное значения функции $\cos$ равны 1 и -1 соответственно;
   - период функции $\cos(kx)$ равен $\frac{2\pi}{k}$;
   - функция $\cos$ является чётной, то есть $\cos(-x) = \cos(x)$.
3. План решения задачи:
   - определите параметры $a$, $b$, $c$, $d$ из условия задачи;
   - подставьте найденные параметры в формулу функции;
