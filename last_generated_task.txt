===ЗАДАЧА===
Решите неравенство: $\log_3(x+5) + \log_3(x-1) \leq 2$.

===РЕШЕНИЕ===
<p>1. Сначала объединим логарифмы в левой части неравенства, используя свойство суммы логарифмов: $\log_a(m) + \log_a(n) = \log_a(m \cdot n)$.<br>
   $$\log_3((x+5)(x-1)) \leq 2$$</p><p>2. Преобразуем неравенство, используя определение логарифма: $\log_a(b) = c$ эквивалентно $b = a^c$.<br>
   $$(x+5)(x-1) \leq 3^2$$<br>
   $$(x+5)(x-1) \leq 9$$</p><p>3. Раскроем скобки и приведём неравенство к стандартному виду:<br>
   $$x^2 + 4x - 5 \leq 9$$<br>
   $$x^2 + 4x - 14 \leq 0$$</p><p>4. Решим полученное квадратное неравенство. Для этого найдём корни соответствующего квадратного уравнения:<br>
   $$x^2 + 4x - 14 = 0$$<br>
   Используем формулу корней квадратного уравнения:<br>
   $$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$<br>
   где $a = 1$, $b = 4$, $c = -14$.<br>
   $$x = \frac{-4 \pm \sqrt{16 + 56}}{2}$$<br>
   $$x = \frac{-4 \pm \sqrt{72}}{2}$$<br>
   $$x = \frac{-4 \pm 6\sqrt{2}}{2}$$<br>
   $$x_1 = -2 + 3\sqrt{2}, \quad x_2 = -2 - 3\sqrt{2}$$</p><p>5. Определим интервалы, на которых выполняется неравенство $x^2 + 4x - 14 \leq 0$. Учитывая, что ветви параболы направлены вверх, неравенство выполняется между корнями:<br>
   $$-2 - 3\sqrt{2} \leq x \leq -2 + 3\sqrt{2}$$</p><p>6. Учтем ОДЗ для логарифмических выражений: $x + 5 > 0$ и $x - 1 > 0$.<br>
   $$x > -5 \quad \text{и} \quad x > 1$$<br>
   Объединяя эти условия, получаем:<br>
   $$x > 1$$</p><p>7. Окончательный ответ — пересечение интервала, полученного из квадратного неравенства, и ОДЗ:<br>
   $$1 < x \leq -2 + 3\sqrt{2}$$</p><p>Ответ: $1 < x \leq -2 + 3\sqrt{2}$</p>

===ПОДСКАЗКИ===
1. В логарифмическом неравенстве $\log_3(x+5) + \log_3(x-1) \leq 2$ требуется найти все его решения. Обратите внимание на два ключевых момента: во-первых, основание логарифма равно 3, а сумма логарифмов не должна превышать 2; во-вторых, необходимо учесть ОДЗ — области допустимых значений для логарифмических выражений.
2. Для решения логарифмического неравенства $\log_3(x+5) + \log_3(x-1) \leq 2$ можно использовать следующие свойства и формулы: 1) свойство суммы логарифмов: $\log_a(m) + \log_a(n) = \log_a(m \cdot n)$; 2) формула перехода от логарифма к степени: $\log_a(b) = c$ эквивалентно $b = a^c$; 3) условие ОДЗ для логарифма: подлогарифмическое выражение должно быть положительно.
3. Для решения логарифмического неравенства $\log_3(x+5) + \log_3(x-1) \leq 2$ выполните следующие действия: 1) Используя свойство суммы логарифмов, запишите $\log_3((x+5)(x-1)) \leq 2$; 2) По свойству равенства логарифмов, получаем $(x+5)(x-1) \leq 3^2 = 9$; 3) Раскройте скобки и приведите к виду $x^2 + 4x - 5 \leq 9$, что преобразуется в квадратное неравенство $x^2 + 4x - 14 \leq 0$; 4) Решите это квадратное неравенство, используя формулу корней, и проверьте корни на принадлежность ОДЗ.
